CookSuggest - スマート料理提案システム

食材の無駄を削減し、創造的な料理体験を提供
冷蔵庫に余っている食材を入力するだけで、栄養バランスと美味しさを考慮した最適なレシピを瞬時に提案します。1000種類以上の厳選されたレシピデータベースから、あなたの食材・調理時間・難易度・アレルギー情報に完全マッチした料理を5つセレクト。

🌟 主要機能:
• AI搭載の食材マッチングアルゴリズム
• 栄養価計算と健康的なバランス提案
• 調理時間・難易度による絞り込み
• アレルギー・食事制限対応フィルター
• 季節の食材を活用した旬のレシピ推奨
• ユーザー評価による品質保証システム
• 買い物リスト自動生成機能

React/PWA・栄養価API・機械学習・レスポンシブデザイン・オフライン対応

地震モニター - 高精度災害情報システム

気象庁API連携による信頼性の高い地震情報配信
日本全国の地震情報を秒単位でリアルタイム監視し、緊急時の迅速な判断をサポート。震度・マグニチュード・震源地を視覚的に把握できる直感的なインターフェースで、防災意識の向上と安全確保に貢献します。

⚡ 主要機能:
• 気象庁公式APIからの即時データ取得
• インタラクティブ地図による震源地可視化
• 震度別カラーコーディング表示
• 地域別絞り込み・履歴検索機能
• プッシュ通知による緊急アラート
• 過去30日間の地震活動統計分析
• 津波警報・注意報の同時表示
• 避難所情報との連携機能

気象庁API・WebSocket・Progressive Web App・プッシュ通知・地図API・データ可視化

社会保険ニュース - 自動情報収集・分析システム

専門家レベルの社会保険情報を自動化で効率配信
厚生労働省・日本年金機構等の公式発表を24時間自動監視し、重要な制度変更や給付金情報をAI技術で要約・分類。社会保険労務士や人事担当者、個人の年金・保険の最新動向把握を強力サポート。

📋 カバー分野:
• 厚生年金制度の改正・給付変更
• 健康保険の保険料率・制度変更
• 雇用保険の失業給付・助成金情報
• 労災保険の認定基準・補償内容
• 介護保険の改正・サービス変更
• マイナンバー関連の社会保険手続き
• 副業・フリーランス向け保険情報
• 企業の社会保険実務への影響分析

Python Web Scraping・自然言語処理・機械学習分類・自動レポート生成・RSS配信

Tidyverse/Tidymodels完全ガイド - R言語データサイエンス学習サイト

データサイエンティストを目指す全ての人へ
R言語の最強パッケージ群「Tidyverse」と機械学習フレームワーク「Tidymodels」を、基礎から実践まで体系的に学べる日本語完全ガイド。全25章、50,000語超の圧倒的ボリュームで、データ分析のプロフェッショナルスキルを習得。

📚 充実のカリキュラム:
• 第1-9章：Tidyverse基礎（dplyr, ggplot2, tidyr等）
• 第10-15章：統計・可視化の応用
• 第16-20章：機械学習（tidymodels, リサンプリング, アンサンブル）
• 第21-23章：時系列分析・テキスト分析・ディープラーニング
• 第24-25章：実践プロジェクト・Kaggle挑戦

初心者からデータサイエンティストまで、あらゆるレベルに対応。Python経験者のR言語移行にも最適。サンプルコード・演習問題・実践例が豊富で、手を動かしながら学べます。

R言語・Tidyverse・Tidymodels・dplyr・ggplot2・機械学習・データ可視化・統計解析・Kaggle
