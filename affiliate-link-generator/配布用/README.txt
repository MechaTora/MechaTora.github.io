===============================================
🛍️ Amazon・楽天アフィリエイトリンク自動生成ツール
===============================================

🚀 **超簡単スタート！**

1. 📄 「インストールと使い方.html」をブラウザで開く
2. 🔧 「簡単インストール.bat」を右クリック→管理者として実行
3. ⏳ インストール完了まで待機
4. 🎯 「start.bat」をダブルクリックして起動

===============================================

📋 **最初にやること**

1. Amazon アソシエイトプログラム登録
   → https://affiliate.amazon.co.jp/

2. 楽天アフィリエイトプログラム登録  
   → https://affiliate.rakuten.co.jp/

3. Google Chrome インストール
   → https://www.google.com/chrome/

===============================================

🎯 **使い方**

1. Excel で商品リスト作成（B列に商品名）
2. ツール起動
3. 設定でアフィリエイトID入力
4. ファイル選択→リンク生成開始
5. 完了！

===============================================

❓ **困ったときは**

- 「インストールと使い方.html」に詳細説明あり
- エラーが出たらログをコピーして相談
- Chrome がないと動作しません

===============================================

⚠️ **注意事項**

- Amazon・楽天の利用規約を遵守してください
- 適切な検索間隔で利用してください（推奨5秒以上）
- 商用利用時は各サービスの規約を確認してください
