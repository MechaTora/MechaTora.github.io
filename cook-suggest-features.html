AI搭載スマート料理提案システム - 機能詳細解説

🎯 CookSuggestとは

CookSuggestは、冷蔵庫に余っている食材から最適なレシピを提案するAI搭載の料理支援アプリケーションです。 食材の無駄を削減し、栄養バランスを考慮した創造的な料理体験を提供します。

🤖AI搭載マッチングシステム

機械学習アルゴリズムによる食材と料理の最適マッチング。ユーザーの嗜好学習機能付き。

📊栄養価自動計算

カロリー、タンパク質、炭水化物、脂質、ビタミン、ミネラルを自動算出。健康管理をサポート。

⚡瞬時検索

1000種類以上のレシピから3秒以内で最適な料理5つを提案。ストレスフリーな料理選択。

🏥アレルギー対応

卵、乳製品、小麦、そば、エビ、カニ、落花生の7大アレルゲン対応フィルター機能。

🔬 レシピマッチングアルゴリズム

AI搭載検索プロセス

1
食材入力解析
入力された食材名の表記揺れを正規化（玉ねぎ/タマネギ/玉葱 → 統一）
2
レシピデータベース検索
1000+ レシピから食材含有率・栄養バランス・調理難易度で1次絞り込み
3
スコアリング計算
食材一致率(40%) + 栄養バランス(30%) + 調理時間(20%) + ユーザー嗜好(10%)
4
多様性最適化
和洋中のバランス、調理法の多様性を考慮し上位5レシピを選定
5
結果出力
栄養価情報・調理時間・難易度・買い足し食材リストと共に提案

🍎 栄養価計算システム

文部科学省「日本食品標準成分表2020年版（八訂）」に基づく正確な栄養価計算を実装。

タンパク質 (g) ±3% 文科省成分表 筋肉量維持・成長サポート
脂質 (g) ±4% 文科省成分表 必須脂肪酸摂取・満腹感

食物繊維 (g) ±5% 文科省成分表 腸内環境改善・便秘解消

⚙️ 技術仕様

  • フロントエンド: React 18.2.0 + TypeScript 4.9
  • 状態管理: Redux Toolkit + RTK Query
  • UI Framework: Material-UI 5.0 + Emotion CSS-in-JS
  • PWA: Workbox 6.0 + Service Worker
  • データベース: IndexedDB (Dexie.js) + LocalStorage
  • 検索エンジン: Fuse.js 6.0 (ファジー検索)
  • 栄養計算API: USDA FoodData Central + 文科省成分表
  • デプロイ: GitHub Pages + CDN (Cloudflare)
