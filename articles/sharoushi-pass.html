社労士試験合格体験記｜働きながら9ヶ月で合格した勉強法と時間管理術

📅 公開日: 2025年1月15日 🔄 最終更新: 2025年1月15日

「働きながら社労士試験に合格できるのか？」——結論から言うと、できます。私は人事の仕事をしながら、2021年11月中旬に勉強を始め、翌2022年（令和4年度）の試験で一発合格しました。しかもこの年は、直近5年でもっとも合格率が低い5.3%。正直に言うと勉強は好きではなく、特に暗記が大の苦手で、高校の世界史ではカタカナの人物名が覚えられず赤点を取ったほどです。そんな私でも、通信制予備校を軸に隙間時間をかき集めて約1,000〜1,100時間を積み上げ、合格までたどり着けました。この記事では、実際にやったこと・使った教材・時間の作り方を、包み隠さずお伝えします。

なぜ社労士資格を取得しようと思ったのか

きっかけは「全体を見る人がいない」という現場の課題

当時、私の会社には顧問社労士がおらず、労務まわりの実務を担当数名でローテーションしながら回していました。専門職的な担当者がいないので、法令の細かい部分の不知からくるケアレスミスも少なくありません。「全体を体系的に見られる人が、一人はいてもいいよな」——そう感じたのが、社労士を目指した最初のきっかけです。

もう一つは、キャリアの先回りです。将来マネジャーを目指すなら労基法の知識は必須ですし、社労士は人事系では最高峰の資格。転職や独立の選択肢にもつながると考えました。

「実務経験あるし余裕でしょ」→ 5%も知らなかった

実は私は社会保険業務を数年やっていて、衛生管理者の資格も持っていました。だから「試験範囲もある程度は知っているだろう」と、正直かなり軽いノリでスタートしました。ところが、ふたを開けてみると——試験範囲の全体の5%も知らなかったのです。実務で触るのはごく一部。安衛法が多少わかる程度で、体系的な知識はほぼゼロからでした。この「勘違い」に早く気づけたのは、結果的に良かったと思います。

私の合格スペック

基本情報

  • 受験年：令和4年度（2022年）※合格率5.3%、直近5年で最低
  • 勉強期間：2021年11月中旬 → 2022年8月（約9ヶ月半）
  • 総勉強時間：約1,000〜1,100時間
  • 受験回数：1回（一発合格）
  • 学習形態：通信制予備校をメイン＋市販教材を併用
  • 仕事：出社7割・月残業30時間前後・通勤は往復2時間半
  • 家庭：30代・未就学児1人・妻はフルタイム勤務

勉強時間の内訳（隙間時間の積み上げ）

まとまった時間を確保するのは現実的に不可能だったので、細切れの隙間時間をスマホで積み上げました。合算するとだいたい以下のペースです。

  • 平日：だいたい3時間
  • 休日：3〜5時間（前半6ヶ月は3時間程度）
  • 直前期：8時間/日

試験結果

私が受けた令和4年度の合格ラインは 選択式27点・択一式44点。合格率5.3%という難しい年でしたが、ここを上回って合格できました。

【試験当日のリアル】
会場はとある大学の窓側の席。行きは豪雨で靴がびしょ濡れになりげんなり……と思ったら、到着した途端に一転して快晴。今度は窓際の木にいたセミが試験中ずっとけたたましく鳴き、イライラしながら午前の問題を解きました。湿度＋マスク＋会場の人口密度で居心地は最悪。午後は時間ちょうどに解き終え、くたくたになって帰宅。翌日の出勤が本当につらかったのを、今でも覚えています。

使用した教材

学習の背骨にしたのは通信制の予備校です。仕事で通学の時間が取れないこと、隙間時間にスマホで進めたいこと、活字だけでは理解しきれず動画講義が欲しかったこと——この3点から通信制を選びました。予備校はたくさんあるので、口コミや各社の特性を比較して1社に絞っています（11月開始で「お試し受講」の時間ももったいないと考え、サクッと決めました）。以下は、その予備校教材を補うために併用した市販教材です。

基本テキスト（併用）

「みんなが欲しかった! 社労士の教科書」（TAC出版）

  • メリット: フルカラーで図解が豊富、初学者でも理解しやすい
  • 使い方: 1周目は通読、2周目以降は問題集と併用
  • 価格: 約4,000円（上下巻セット）

問題集

「みんなが欲しかった! 社労士の問題集」（TAC出版）

  • メリット: テキストと完全連動、解説が詳しい
  • 使い方: 3周以上繰り返す（間違えた問題は5周以上）

過去問題集

「社労士 過去10年本試験問題集」（成美堂出版）

  • メリット: 過去10年分が収録、本番の難易度を体感できる
  • 使い方: 試験3ヶ月前から、毎週1年分を時間を測って解く

その他

  • YouTube講義動画：「社労士24」（資格の大原）の無料サンプル動画
  • スマホアプリ：「社労士 一問一答」（通勤時間に活用）

使い分けの考え方はシンプルで、予備校の動画講義で「理解の土台」を作り、市販の問題集・過去問・アプリで「隙間時間のアウトプット」を回す、という二本立てです。教材そのものより、「どの細かさで暗記すべきか」「何を併用すると効くか」を早く見極められるかが、タイパを大きく左右しました。

📝 教材選びの詳細や科目別の暗記法・語呂合わせまで含めた「最短ルートの対策メソッド」は、私のnoteにまとめています（累計50部以上・合格報告も頂いています）。→ 暗記苦手でも9か月で一発合格！社労士試験の対策メソッド

9ヶ月の勉強スケジュール

11月〜1月（基礎インプット期）

目標：全科目のテキストを1周する

  • 平日: 2時間（朝1時間 + 夜1時間）
  • 休日: 5時間
  • やったこと: テキストを読む、理解が難しい箇所はYouTubeで補足

この時期は「完璧に理解しよう」とせず、「全体像を掴む」ことを重視しました。労働法と社会保険法の全体構造を理解するのが目標です。

2月〜4月（問題演習期1）

目標：問題集を2周する

  • 平日: 2.5時間（朝1時間 + 夜1.5時間）
  • 休日: 6時間
  • やったこと: 問題集を解く → 間違えた問題はテキストに戻る

1周目の正答率は50%程度で、心が折れそうになりました。しかし、「間違えることで記憶に定着する」と割り切り、淡々と進めました。

5月〜6月（問題演習期2 + 弱点補強）

目標：問題集3周目 + 弱点科目の集中学習

  • 平日: 3時間（朝1時間 + 夜2時間）
  • 休日: 7時間
  • やったこと: 間違えた問題を重点的に復習、一般常識の対策開始

この時期から「間違いノート」を作成し、繰り返し間違える論点をまとめました。

7月〜8月（過去問演習 + 総仕上げ）

目標：過去問で合格点を安定して取る

  • 平日: 3.5時間（朝1時間 + 夜2.5時間）
  • 休日: 8時間
  • やったこと: 過去問を時間を測って解く、模試を受験（2回）

試験1ヶ月前からは、毎週日曜日に過去問1年分を本番と同じ時間で解きました。時間配分の感覚を掴むのが目的です。

科目別攻略法

労働基準法・労働安全衛生法（配点大、比較的易しい）

攻略法：

  • 労働基準法は確実に満点を狙う（基本中の基本）
  • 労働時間、休日、年次有給休暇は頻出なので完璧に
  • 条文の数字（36協定の限度時間など）は暗記カードで覚える

労災保険法・雇用保険法（配点大、実務でも使う）

攻略法：

  • 給付の種類と要件を表にまとめる
  • 数字の暗記が重要（待期期間、給付率など）
  • 実務経験者は有利な科目

健康保険法・厚生年金保険法・国民年金法（配点最大、最重要）

攻略法：

  • 年金3法は勉強時間の40%を割く（最重要）
  • 老齢年金の計算式は完璧に理解する
  • 被保険者の種別、適用除外など基本事項を確実に
  • 健康保険と国民健康保険の違いを明確に

一般常識（労働・社保）（配点小、範囲広い、最難関）

攻略法：

  • 深追いしない（コスパが悪い）
  • 労働経済白書、厚生労働白書の最新データをチェック
  • 統計数値（失業率、有効求人倍率など）は試験直前に暗記
  • 基準点（3/5点）さえクリアすれば良いと割り切る

私はこの科目で3点しか取れず、基準点ギリギリでした。範囲が広すぎて対策が難しいため、他の科目で点数を稼ぐ戦略が重要です。

働きながら勉強時間を確保する方法

朝活の活用（最重要）

平日は朝5時に起床し、6時まで勉強しました。朝は頭が冴えていて、集中力が高いです。夜は疲れて効率が落ちるため、朝に重点を置きました。

通勤時間の活用

片道40分の通勤時間は、スマホアプリで一問一答を解きました。1日往復で80分 = 約1ヶ月で40時間の勉強時間になります。

昼休みの15分

ランチ後の15分は、前日の復習に使いました。「昨日間違えた問題を3問解く」だけでも、記憶の定着に効果的です。

休日は図書館へ

家では誘惑が多いため、土日は図書館で勉強しました。朝9時〜18時まで（昼休み1時間）で8時間確保できます。

合格するために必要なマインドセット

1. 完璧主義を捨てる

社労士試験は満点を取る試験ではなく、「基準点をクリアする」試験です。全科目で7割取れれば合格できます。わからない問題は深追いせず、次に進む勇気が必要です。

2. 継続が最強

「毎日2時間」を9ヶ月続けることが、「週末に10時間」を数回やるより効果的です。記憶の定着には「反復」が不可欠です。

3. アウトプット中心

テキストを読むだけでは合格できません。問題を解いて、間違えて、復習する。このサイクルを何度も回すことが重要です。

4. 模試は必ず受ける

本番と同じ時間配分、緊張感を体験できるのは模試だけです。私は2回受けて、時間配分のミスを修正できました。

合格後のキャリアへの影響

実務での変化

  • 自信を持って判断できる：法律の根拠を説明できるようになった
  • 相談される機会が増えた：社内で「労務のプロ」として認識された
  • 業務の幅が広がった：就業規則改定、助成金申請など高度な業務を担当

キャリアの選択肢が増えた

  • 社労士資格を活かして、人事コンサルティング会社への転職も視野に
  • 将来的には独立開業も選択肢の一つ
  • 副業で労務相談を受けるなど、複数の収入源を確保

まとめ

社労士試験は難関資格ですが、正しい勉強法と継続力があれば、働きながらでも合格可能です。このページのポイントをまとめます：

  • 勉強期間：9ヶ月、約800時間（働きながら）
  • 教材：独学（市販テキスト + 問題集 + 過去問）で総額2万円
  • 科目別戦略：年金3法に40%、一般常識は基準点狙い
  • 時間確保：朝活 + 通勤時間 + 休日図書館で1日3〜8時間
  • 合格の鍵：完璧主義を捨て、継続し、アウトプット中心の学習

これから社労士試験に挑戦する方へ。合格率6〜7%という数字に怯む必要はありません。正しい方法で努力を続ければ、必ず合格できます。応援しています！
