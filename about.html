🌟 MechaTora（メカトラ）について

「日常の小さな不便を解決する」をモットーに、実用的なWebアプリケーションを開発・運営しています。料理で何を作るか迷った時、地震情報をリアルタイムで知りたい時、効率的に学習したい時など、具体的な課題に対応したツールを無料で提供。2025年4月の開設以来、継続的にサービスを運営・改善しています。

👨‍💻 運営者プロフィール

🏆 資格・認定

  • 社会保険労務士 - 社会保険・労働法の専門家
  • 第一種衛生管理者 - 労働安全衛生の国家資格
  • 応用情報技術者 - 高度IT人材の証明
  • 基本情報技術者 - IT基礎知識の国家資格
  • 簿記2級 - 企業会計の実務知識
  • G検定（JDLA Deep Learning for GENERAL） - AI・機械学習の基礎知識

💼 経歴・専門性

Web開発歴12年のシニアフルスタックエンジニア

  • フロントエンド：JavaScript, TypeScript, React, Vue.js, HTML5, CSS3
  • バックエンド：Python, Node.js, PHP, データベース設計
  • インフラ：AWS, GitHub Pages, Netlify, Render.com
  • データサイエンス：R言語（Tidyverse）, Python（Pandas, NumPy）, 機械学習
  • 専門分野：社会保険制度、労務管理、人事DX、PWA開発

🎯 開発スタンス

文系人事職からWeb開発の世界へ転身。「非エンジニアでも、アイデアと情熱があれば価値あるサービスを作れる」ことを実証するため、Claude Code（AI支援開発）を活用した「バイブコーディング」スタイルで開発を進めています。

完璧主義を捨て、「80点で公開する勇気」を持ち、ユーザーフィードバックを重視した継続的改善を行っています。

🌱 開発の歴史

2022年4月

MechaTora立ち上げ。最初のアプリ「CookSuggest」をリリース。

2023年

地震モニター、暗記カードアプリなど実用ツールを順次追加。PWA対応を開始。

2024年

癒し系コンテンツ（暖炉、オーロラ、クラゲ）を追加。科学的根拠に基づく価値訴求を強化。

2025年9月

独自ドメイン「mechatora.com」取得。R言語・Tidyverse解説サイト制作開始。Google AdSense審査に挑戦中。
